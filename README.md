## Installation

This package can be installed with the following command:

```
pip install git+github.com/RackReaver/finpack.git
```

## Running Tests and Checking Coverage

### Run tests and check for test coverage

```
$ coverage run -m pytest
```

### Show test coverage

```
$ coverage report
```
